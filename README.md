# JxDroid
Симулятор Android устройства прямо в раузере!
## Запуск
Для запуска сайта вы можете использовать один из вариантов:
### 1. Запуск через GitHub Pages
- Стабильная версия - [jxstab.github.io](https://jxdroid.github.io/)
- Beta - [mansar1337.github.io/jxdroid.github.io](https://mansar1337.github.io/jxdroid.github.io/)
### 2. Запуск локально
- ```1.``` Скачайте исходник нажав [сюда](https://github.com/jxdroid/jxdroid.github.io/archive/refs/heads/main.zip)
- ```2.``` Распакуйте скачанный архив
- ```3.``` Откройте ```index.html``` в браузере
